Site report for the website Alan biju's Portfolio

Introduction

Hi my name is Alan Biju and this is my portfolio website.
This website is a project for the web development course at the university of the Northampton.
The website is a simple portfolio website that showcases , my details, my projects, contact information, site report and with a small video demostration.
The website is designed to be simple and easy to navigate.

learning experience

Throughout the module I emberked on a journey to learn web development from scratch.
I have learned a lot from this project.
I have learned how to create a website from scratch using HTML and CSS.
I have learned how to use flexbox and grid to create a responsive layout.
I have learned how to use media queries to make the website responsive.
I have learned how to use JavaScript to create a hamburger menu.
I have learned how to use git and github to manage my project.
I gained a lot of experience from this project and I am happy with the end result, which helped me gain confidence for the future.

The debugging process was like a rollercoaster ride, I had to go through a lot of errors and bugs(for example, like typos and forgotten to close the tags and so on) to get the website to work properly.
I also learned the importance of responsive design ,testing my site on different screen sizes and fixing layout shifts with media queries.

site development and design decisions

For the UI, I wanted a clean and modern aesthetic.
I chose a arial sans-serif font family for all text in my website as it was a intersting and elegant font.
I used a color scheme of black, white and blue for the website with background image as wonderful gif which related to technology, I liked that one because of that circular movement which shows the link between human life and technology.
I used a hamburger menu for the navigation on mobile devices.
I used a flexbox for the layout of the website.
I used a grid for the project section to display the projects in a grid layout.
I used a media query to make the website responsive.
I opted for simple and clean navigation bar and section divisions.

I experimented with color plaettes using colorhunt.co and coolors.co to find the right color scheme for the website.
I ensured that the website was accessible and easy to navigate.
I used the colours like turquoise,marigold and amber color which represent the truth about love from diverse sides(like enduring love, warm love,joyous love)
And used interactive elements like hover effects and transitions to make the website more engaging.

Reflection

I am happy with the end result of the website.
Overall, I am happy with the design and layout of the website.
This project taught me resilience and problem solving
Each challenge I faced was a learning opportunity.
I now appreciate the importance of responsive design and accessibility.
Also this encourage me to do more work in complex and dynamic sites in the future.

I'm pround of the progress I have made and gateful for the resources and feedback that guided me through this project.
thanks for everything.
